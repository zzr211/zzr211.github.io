Hugo搭建静态博客网站

# 1. 安装 hugo

# （1）下载 hugo

选择下载 hugo 的 extend 版本，我下载的是 hugo_extended_0.96.0_Windows-64bit.zip

地址：https://github.com/gohugoio/hugo/releases

解压得到 hugo.exe，将 hugo.exe 复制到本地建站目录，比如：

C:\Users\ZR\Documents\GitHub\hugo

# （2）添加至 Path

将 hugo.exe 所在目录添加至系统环境变量（我的电脑 - 属性 - 高级系统设置 - 环境变量 - 系统变量 - Path - 浏览添加）

或者将 hugo 下载至命令行所在文件夹，然后将下载的 hugo 文件夹加入 Path 中

# （1）选择 hugo 网站目录

# （2）执行命令新建 hugo 网站

新建的网站目录大概包含这些目录 ——C:\Users\ZR\Documents\GitHub\hugo\config.toml、hugo.exe、archetypes、content、data、layouts、static、themes

  • archetypes: 储存 .md 的模板文件，类似于 hexo 的 scaffolds ，该文件夹的优先级高于主题下的 /archetypes 文件夹
  • config.toml: 配置文件
  • content: 储存网站的所有内容，类似于 hexo 的 source
  • data: 储存数据文件供模板调用，类似于 hexo 的 source/_data
  • layouts: 储存 .html 模板，该文件夹的优先级高于主题下的 /layouts 文件夹
  • static: 储存图片，css,js 等静态文件，该目录下的文件会直接拷贝到 /public ，该文件夹的优先级高于主题下的 /static 文件夹
  • themes: 储存主题
  • public: 执行 hugo 命令后，储存生成的静态文件
