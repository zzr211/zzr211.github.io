中国传统色配色网站

发表于2022-08-09|更新于2023-08-18
|阅读量:

# 1. 中国色

http://zhongguose.com/

收录有 560 种中国传统色，颜色信息相当完整，包括色值、色名、色谱、中国色名综览等，并根据颜色名称提供了 CMYK 与 RPG 两者的颜色色值。

该网站收录的颜色全部出自中国社科院情报编委会名词室编撰的《色谱》，全部都是几千年来流传下来的中国色，绝不是日本传统色（涵盖了 250 种日本传统颜色的网站）NIPPON COLORS 所能比拟的，希望大家能够清楚这一点。

# 2. 中国传统颜色在线手册

https://colors.ichuantong.cn/

  • 项目开源（地址 https://github.com/zerosoul/chinese-colors）

  • UI 有自己的设计

  • 每一种颜色都有对应的诗词并导出图片

# 3.（没有名字）

http://www.2kil.com/

没啥特别的

# 4. 中国色彩

https://chinese-colors.heyfe.org/

  • UI 有自己的设计

  • 提供网站配色示例预览，以选择的颜色为基础生成一套网站主题配色，包括字色、高亮字色、背景色、边框色等。

# 5. 中国色颜色色谱

https://coolfishstudio.github.io/cfs-color/

  • 界面简单易用
  • 点击即复制色值

# 6. 在线颜色选择器 | RGB 颜色查询对照表

提供颜色值的计算工具，以及几种颜色选择的工具，方便大家选择颜色，并将颜色转换成自己想要的格式。

http://tools.jb51.net/static/colorpicker/
